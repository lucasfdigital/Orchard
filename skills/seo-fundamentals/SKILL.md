# Fundamentos de SEO

> Princípios para visibilidade em motores de busca.

---

## 1. Framework E-E-A-T

| Princípio | Sinais |
| :--- | :--- |
| **Experiência** | Conhecimento de primeira mão, exemplos reais |
| **Expertise (Especialidade)** | Credenciais, profundidade de conhecimento |
| **Autoridade** | Backlinks, menções, reconhecimento da indústria |
| **Confiança (Trust)** | HTTPS, transparência, informação precisa |

---

## 2. Core Web Vitals

| Métrica | Objetivo | O que mede |
| :--- | :--- | :--- |
| **LCP** | < 2.5s | Performance de carregamento (Largest Contentful Paint) |
| **INP** | < 200ms | Interatividade (Interaction to Next Paint) |
| **CLS** | < 0.1 | Estabilidade visual (Cumulative Layout Shift) |

---

## 3. Princípios de SEO Técnico

### Estrutura do Site

| Elemento | Propósito |
| :--- | :--- |
| Sitemap XML | Ajuda no rastreamento (crawling) |
| robots.txt | Controla o acesso |
| Tags canônicas | Previne conteúdo duplicado |
| HTTPS | Sinal de segurança |

### Performance

| Fator | Impacto |
| :--- | :--- |
| Velocidade da página | Core Web Vital |
| Amigável para mobile | Fator de ranking |
| URLs limpas | Facilidade de rastreamento |

---

## 4. Princípios de SEO de Conteúdo

### Elementos da Página

| Elemento | Melhor Prática |
| :--- | :--- |
| Tag de Título | 50-60 caracteres, palavra-chave no início |
| Meta descrição | 150-160 caracteres, persuasiva |
| H1 | Um por página, palavra-chave principal |
| H2-H6 | Hierarquia lógica |
| Texto Alt (imagens) | Descritivo, sem excesso de palavras-chave |

### Qualidade do Conteúdo

| Fator | Importância |
| :--- | :--- |
| Profundidade | Cobertura abrangente |
| Atualidade | Atualizações regulares |
| Singularidade | Valor original |
| Legibilidade | Escrita clara |

---

## 5. Tipos de Marcação de Schema

| Tipo | Uso |
| :--- | :--- |
| Article | Posts de blog, notícias |
| Organization | Informações da empresa |
| Person | Perfis de autores |
| FAQPage | Conteúdo de perguntas e respostas |
| Product | E-commerce |
| Review | Avaliações e notas |
| BreadcrumbList | Navegação (trilhas) |

---

## 6. Diretrizes de Conteúdo de IA

### O que o Google procura

| ✅ FAÇA | ❌ NÃO FAÇA |
| :--- | :--- |
| Rascunho de IA + Edição humana | Publicar conteúdo de IA bruto |
| Adicionar insights originais | Copiar sem agregar valor |
| Revisão de especialistas | Pular checagem de fatos |
| Seguir o E-E-A-T | Excesso de palavras-chave (stuffing) |

---

## 7. Fatores de Ranking (Priorizados)

| Prioridade | Fator |
| :--- | :--- |
| 1 | Conteúdo relevante e de qualidade |
| 2 | Backlinks de sites com autoridade |
| 3 | Experiência da página (Core Web Vitals) |
| 4 | Otimização para dispositivos móveis |
| 5 | Fundamentos de SEO técnico |

---

| :--- | :--- |

> **Lembre-se:** SEO é um jogo de longo prazo. Conteúdo de qualidade + excelência técnica + paciência = resultados.
